# CMSC-207-Mini-Project
Personal Web Profile Mini Project for CMSC 207

A simple personal profile website built with HTML, CSS, and JavaScript.  


**Features**
- Clean single-page profile layout
- Sections for **About Me**, **Hobbies**, and **Contact**
- Interactive **Fun Fact** button (show/hide behavior via JavaScript)
- External links to GitHub and LinkedIn
- Responsive structure using semantic HTML

**Tech Stack**
- HTML5
- CSS3
- JavaScript (Vanilla)

**Project Structure**
- `index.html` — main page content and layout
- `styles.css` — site styling
- `script.js` — interactive behavior
- `assets/picture.jpg` — profile image

**Run Locally**
1. Open the project folder in VS Code.
2. Open `index.html` in a browser  
   **or** use the VS Code **Live Server** extension.
